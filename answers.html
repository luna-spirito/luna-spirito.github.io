Конечно, вот подробные ответы на экзаменационные вопросы, составленные на основе предоставленных лекций.

1. Вывод уравнения колебаний струны. Постановка краевых задач и задачи Коши для уравнения колебаний струны.

(Источник: Лекция 1, раздел 1; Лекция 2, раздел 4)

Рассмотрим идеальную гибкую струну. Предположим, что: 1. Струна однородна, её масса на единицу длины в состоянии покоя постоянна и равна \rho. 2. Натяжение струны T велико, и оно не меняется при колебаниях. 3. Колебания являются малыми и поперечными, т.е. точки струны смещаются только в одной плоскости (плоскости XOY) перпендикулярно оси OX. 4. Струна не сопротивляется изгибу.

Пусть u(t, x) — отклонение точки струны с абсциссой x от положения равновесия в момент времени t.

Выделим малый участок струны между точками x_1 и x_2. Пусть \alpha_1 и \alpha_2 — углы, образованные касательными к струне в этих точках с осью OX.

Поскольку колебания малые, \alpha \approx \sin \alpha \approx \tan \alpha = u_x(t,x).

  1. Проекция сил на ось OX: Силы натяжения, действующие на концах участка, равны T_1 и T_2. -T_1 \cos \alpha_1 + T_2 \cos \alpha_2 = 0 Так как \cos \alpha \approx 1 для малых углов, получаем T_1 \approx T_2 \approx T. Сила натяжения постоянна вдоль всей струны.

  2. Проекция сил на ось OY: Результирующая сила в вертикальном направлении складывается из вертикальных компонент сил натяжения и внешней распределенной силы f(t, x): F_y = -T \sin \alpha_1 + T \sin \alpha_2 + \int_{x_1}^{x_2} f(t, \xi) d\xi Используя приближение \sin \alpha \approx \tan \alpha = u_x, получаем: F_y \approx T(u_x(t, x_2) - u_x(t, x_1)) + f(t, \xi^*)(x_2 - x_1) где \xi^* \in [x_1, x_2].

  3. Второй закон Ньютона: Масса участка струны m \approx \rho (x_2 - x_1) = \rho \Delta x. Ускорение равно u_{tt}(t, x). m \cdot a = F_y \implies \rho \Delta x \cdot u_{tt}(t, x) \approx T(u_x(t, x_2) - u_x(t, x_1)) + f(t, \xi^*) \Delta x Делим на \Delta x и устремляем \Delta x \to 0: \rho u_{tt}(t, x) = T \lim_{\Delta x \to 0} \frac{u_x(t, x+\Delta x) - u_x(t, x)}{\Delta x} + f(t, x) \rho u_{tt} = T u_{xx} + f(t, x) Окончательно, вводя a^2 = T/\rho, получаем уравнение колебаний струны: u_{tt} = a^2 u_{xx} + f(t, x)

Постановка задач

  1. Задача Коши (для бесконечной струны): Найти решение уравнения u_{tt} = a^2 u_{xx} + f(t, x) для x \in \mathbb{R}, t > 0, удовлетворяющее начальным условиям: \begin{cases} u(0, x) = \varphi(x) & \text{(начальное положение)} \\ u_t(0, x) = \psi(x) & \text{(начальная скорость)} \end{cases}

  2. Краевые задачи (для ограниченной струны x \in [0, L]): Найти решение уравнения u_{tt} = a^2 u_{xx} + f(t, x) для x \in (0, L), t > 0, с начальными условиями и граничными условиями.

    • Первая краевая задача (закрепленные концы): u(t, 0) = g_1(t), \quad u(t, L) = g_2(t)
    • Вторая краевая задача (свободные концы): u_x(t, 0) = h_1(t), \quad u_x(t, L) = h_2(t)
    • Третья краевая задача (упругое закрепление): (\alpha_1 u_x + \beta_1 u)|_{x=0} = \mu_1(t), \quad (\alpha_2 u_x + \beta_2 u)|_{x=L} = \mu_2(t)

2. Формула Даламбера. Существование решения задачи Коши для уравнения колебаний струны.

(Источник: Лекция 1, раздел 2)

Рассмотрим задачу Коши для однородного уравнения колебаний струны (f=0): \begin{cases} u_{tt} = a^2 u_{xx}, \quad t>0, x \in \mathbb{R} \\ u(0, x) = \varphi(x) \\ u_t(0, x) = \psi(x) \end{cases} Для вывода решения используем метод характеристик. Характеристическое уравнение имеет вид (dx)^2 - a^2(dt)^2 = 0, откуда dx = \pm adt. Характеристики — это прямые x \pm at = const.

Сделаем замену переменных: \xi = x + at, \eta = x - at. Тогда u_{tt} - a^2 u_{xx} = -4a^2 u_{\xi\eta} = 0, откуда u_{\xi\eta} = 0. Интегрируя последовательно, получаем общее решение: u(t, x) = F(\xi) + G(\eta) = F(x+at) + G(x-at) где F, G — произвольные дважды дифференцируемые функции.

Подставим начальные условия: 1. u(0, x) = F(x) + G(x) = \varphi(x) 2. u_t(0, x) = aF'(x) - aG'(x) = \psi(x) \implies F'(x) - G'(x) = \frac{1}{a}\psi(x)

Интегрируя второе уравнение, получаем F(x) - G(x) = \frac{1}{a} \int_{x_0}^x \psi(s)ds + C. Решая систему для F(x) и G(x), находим: F(x) = \frac{1}{2}\varphi(x) + \frac{1}{2a}\int_{x_0}^x \psi(s)ds + \frac{C}{2} \\ G(x) = \frac{1}{2}\varphi(x) - \frac{1}{2a}\int_{x_0}^x \psi(s)ds - \frac{C}{2} Подставляя x+at и x-at в качестве аргументов, получаем: u(t, x) = \frac{\varphi(x+at) + \varphi(x-at)}{2} + \frac{1}{2a}\left( \int_{x_0}^{x+at} \psi(s)ds - \int_{x_0}^{x-at} \psi(s)ds \right) u(t, x) = \frac{\varphi(x+at) + \varphi(x-at)}{2} + \frac{1}{2a}\int_{x-at}^{x+at} \psi(s)ds Это и есть формула Даламбера для однородного уравнения.

Для неоднородного уравнения с нулевыми начальными условиями решение дается принципом Дюамеля, и общая формула Даламбера имеет вид: u(t, x) = \frac{\varphi(x+at) + \varphi(x-at)}{2} + \frac{1}{2a}\int_{x-at}^{x+at} \psi(\xi)d\xi + \frac{1}{2a}\int_0^t \int_{x-a(t-\tau)}^{x+a(t-\tau)} f(\tau, \xi) d\xi d\tau

Теорема существования: Пусть \varphi(x) \in C^2(\mathbb{R}), \psi(x) \in C^1(\mathbb{R}) и f(t,x) \in C^1([0, +\infty) \times \mathbb{R}). Тогда задача Коши имеет единственное решение класса C^2([0, +\infty) \times \mathbb{R}), и это решение задается формулой Даламбера. (Источник: Лекция 1, стр. 4, Теорема)

3. Решение с помощью формулы Даламбера краевых задач для полуограниченной струны.

(Источник: Лекция 2, раздел 3)

Рассмотрим струну, занимающую полуось x > 0. Решение ищется с помощью метода отражения (продолжения).

1. Первая краевая задача (закрепленный конец): u(t, 0) = 0 Задача: \begin{cases} u_{tt} = a^2 u_{xx} + f(t,x), \quad t > 0, x > 0 \\ u(0, x) = \varphi(x), \quad u_t(0, x) = \psi(x) \\ u(t, 0) = 0 \end{cases} Продолжим начальные данные и правую часть на всю ось \mathbb{R} нечетным образом: \tilde{\varphi}(x) = \begin{cases} \varphi(x), & x \ge 0 \\ -\varphi(-x), & x < 0 \end{cases} \quad \tilde{\psi}(x) = \begin{cases} \psi(x), & x \ge 0 \\ -\psi(-x), & x < 0 \end{cases} \quad \tilde{f}(t,x) = \begin{cases} f(t,x), & x \ge 0 \\ -f(t,-x), & x < 0 \end{cases} Теперь решаем задачу Коши для бесконечной струны с продолженными данными \tilde{\varphi}, \tilde{\psi}, \tilde{f} по формуле Даламбера. Полученное решение \tilde{u}(t,x) при x=0 будет равно: \tilde{u}(t, 0) = \frac{\tilde{\varphi}(at) + \tilde{\varphi}(-at)}{2} + \dots = \frac{\varphi(at) - \varphi(at)}{2} + \dots = 0 Аналогично, интегралы от нечетных функций по симметричным интервалам равны нулю. Таким образом, граничное условие выполняется автоматически. Решение исходной задачи есть сужение \tilde{u}(t,x) на x > 0.

2. Вторая краевая задача (свободный конец): u_x(t, 0) = 0 Задача аналогична, но с граничным условием u_x(t, 0) = 0. В этом случае данные продолжаются четным образом: \hat{\varphi}(x) = \begin{cases} \varphi(x), & x \ge 0 \\ \varphi(-x), & x < 0 \end{cases} \quad \hat{\psi}(x) = \begin{cases} \psi(x), & x \ge 0 \\ \psi(-x), & x < 0 \end{cases} \quad \hat{f}(t,x) = \begin{cases} f(t,x), & x \ge 0 \\ f(t,-x), & x < 0 \end{cases} Решение \hat{u}(t,x) задачи Коши с этими данными будет иметь производную по x: \hat{u}_x(t,x) = \frac{1}{2}(\hat{\varphi}'(x+at) + \hat{\varphi}'(x-at)) + \dots Поскольку производная четной функции нечетна, \hat{\varphi}' и \hat{\psi} нечетны. Тогда: \hat{u}_x(t, 0) = \frac{1}{2}(\hat{\varphi}'(at) + \hat{\varphi}'(-at)) + \dots = \frac{1}{2}(\hat{\varphi}'(at) - \hat{\varphi}'(at)) + \dots = 0. Граничное условие выполняется.

4. Метод разделения переменных для ограниченной струны.

(Этот вопрос не разобран в лекциях. Приводится стандартное решение.)

Рассмотрим однородную задачу для струны с закрепленными концами (x \in [0,L]): \begin{cases} u_{tt} = a^2 u_{xx} \\ u(0, x) = \varphi(x), \quad u_t(0, x) = \psi(x) \\ u(t, 0) = 0, \quad u(t, L) = 0 \end{cases} Ищем решение в виде u(t, x) = T(t)X(x). Подставляя в уравнение: T''(t)X(x) = a^2 T(t)X''(x) \implies \frac{T''(t)}{a^2 T(t)} = \frac{X''(x)}{X(x)} = -\lambda где -\lambda — константа разделения.

1. Пространственная часть (задача Штурма-Лиувилля): X''(x) + \lambda X(x) = 0, \quad X(0)=0, X(L)=0 Нетривиальные решения существуют только при \lambda > 0. Общее решение: X(x) = C_1 \cos(\sqrt{\lambda}x) + C_2 \sin(\sqrt{\lambda}x). Из X(0)=0 следует C_1=0. Из X(L)=0 следует C_2 \sin(\sqrt{\lambda}L) = 0. Так как C_2 \neq 0 (иначе решение тривиально), то \sqrt{\lambda}L = n\pi для n=1,2,\dots. Собственные значения и функции: \lambda_n = \left(\frac{n\pi}{L}\right)^2, \quad X_n(x) = \sin\left(\frac{n\pi x}{L}\right)

2. Временная часть: T_n''(t) + a^2 \lambda_n T_n(t) = 0 \implies T_n''(t) + \left(\frac{an\pi}{L}\right)^2 T_n(t) = 0 Решение: T_n(t) = A_n \cos\left(\frac{an\pi t}{L}\right) + B_n \sin\left(\frac{an\pi t}{L}\right).

3. Общее решение: По принципу суперпозиции, общее решение есть сумма частных решений: u(t, x) = \sum_{n=1}^{\infty} \left(A_n \cos\left(\frac{an\pi t}{L}\right) + B_n \sin\left(\frac{an\pi t}{L}\right)\right) \sin\left(\frac{n\pi x}{L}\right)

4. Нахождение коэффициентов: Коэффициенты A_n и B_n определяются из начальных условий с помощью рядов Фурье. - u(0, x) = \sum_{n=1}^\infty A_n \sin\left(\frac{n\pi x}{L}\right) = \varphi(x) \implies A_n = \frac{2}{L}\int_0^L \varphi(x)\sin\left(\frac{n\pi x}{L}\right)dx - u_t(0, x) = \sum_{n=1}^\infty B_n \frac{an\pi}{L} \sin\left(\frac{n\pi x}{L}\right) = \psi(x) \implies B_n \frac{an\pi}{L} = \frac{2}{L}\int_0^L \psi(x)\sin\left(\frac{n\pi x}{L}\right)dx B_n = \frac{2}{an\pi}\int_0^L \psi(x)\sin\left(\frac{n\pi x}{L}\right)dx

5. Первая и вторая краевые задачи для волнового уравнения. Энергетическое равенство. Единственность решений.

(Источник: Лекция 2, разделы 4, 5)

Пусть \Omega \subset \mathbb{R}^n — ограниченная область. Рассмотрим цилиндр \omega_T = \Omega \times (0, T). Первая краевая задача (Дирихле): \begin{cases} u_{tt} = a^2 \Delta u + f(t,x), \quad (t,x) \in \omega_T \\ u|_{\partial\Omega \times [0,T]} = \chi(t,x) \\ u(0,x) = \varphi(x), \quad u_t(0,x) = \psi(x) \end{cases} Вторая краевая задача (Неймана): \begin{cases} u_{tt} = a^2 \Delta u + f(t,x), \quad (t,x) \in \omega_T \\ \frac{\partial u}{\partial \nu}|_{\partial\Omega \times [0,T]} = \chi(t,x) \\ u(0,x) = \varphi(x), \quad u_t(0,x) = \psi(x) \end{cases}

Энергетическое равенство Теорема: Пусть u \in C^2(\bar{\omega}_T) является решением однородного волнового уравнения u_{tt} = a^2\Delta u с однородными граничными условиями (u|_{\partial\Omega}=0 или \frac{\partial u}{\partial \nu}|_{\partial\Omega}=0). Тогда для энергии системы E(t) = \frac{1}{2} \int_\Omega (u_t^2(t,x) + a^2 |\nabla u(t,x)|^2) dx справедливо энергетическое равенство: E(t) = E(0) для всех t \in [0,T].

Доказательство: \frac{dE}{dt} = \int_\Omega (u_t u_{tt} + a^2 \nabla u \cdot \nabla u_t) dx Заменим u_{tt} на a^2\Delta u: \frac{dE}{dt} = \int_\Omega (a^2 u_t \Delta u + a^2 \nabla u \cdot \nabla u_t) dx = a^2 \int_\Omega (u_t \Delta u + \nabla u \cdot \nabla u_t) dx По формуле Грина: \int_\Omega v\Delta u \,dx = \int_{\partial\Omega} v \frac{\partial u}{\partial \nu} d\sigma - \int_\Omega \nabla v \cdot \nabla u \,dx. Применим её с v=u_t: \int_\Omega u_t \Delta u \,dx = \int_{\partial\Omega} u_t \frac{\partial u}{\partial \nu} d\sigma - \int_\Omega \nabla u_t \cdot \nabla u \,dx Подставляя обратно в \frac{dE}{dt}: \frac{dE}{dt} = a^2 \left( \int_{\partial\Omega} u_t \frac{\partial u}{\partial \nu} d\sigma - \int_\Omega \nabla u_t \cdot \nabla u \,dx + \int_\Omega \nabla u \cdot \nabla u_t \,dx \right) = a^2 \int_{\partial\Omega} u_t \frac{\partial u}{\partial \nu} d\sigma В случае задачи Дирихле u|_{\partial\Omega}=0 \implies u_t|_{\partial\Omega}=0. В случае задачи Неймана \frac{\partial u}{\partial \nu}|_{\partial\Omega}=0. В обоих случаях подынтегральное выражение равно нулю, следовательно, \frac{dE}{dt}=0, и E(t) = const = E(0).

Единственность решений: Пусть u_1 и u_2 — два решения одной и той же краевой задачи. Их разность w = u_1 - u_2 удовлетворяет однородному уравнению с нулевыми начальными и граничными условиями. E_w(0) = \frac{1}{2} \int_\Omega (w_t^2(0,x) + a^2 |\nabla w(0,x)|^2) dx = 0. Из энергетического равенства E_w(t) = E_w(0) = 0 для всех t. Так как подынтегральное выражение в E_w(t) неотрицательно, то u_t(t,x) \equiv 0 и \nabla u(t,x) \equiv 0. Это значит, что w(t,x) есть константа. Но так как w(0,x)=0, то w(t,x) \equiv 0, и u_1 \equiv u_2. Решение единственно.

6. Задача Коши для волнового уравнения. Энергетическое неравенство. Теорема единственности решения задачи Коши.

(Источник: Лекция 3, раздел 6)

Рассмотрим задачу Коши для однородного волнового уравнения в \mathbb{R}^n: \begin{cases} u_{tt} = a^2 \Delta u, \quad t>0, x \in \mathbb{R}^n \\ u(0, x) = \varphi(x), \quad u_t(0, x) = \psi(x) \end{cases} Решение в точке (t_0, x_0) зависит только от начальных данных в шаре |x-x_0| \le at_0. Это область зависимости. Обратно, начальные данные в точке (0, y) влияют на решение в конусе влияния |x-y| \le at.

Энергетическое неравенство Рассмотрим характеристический конус K = \{(t,x) : 0 < t < t_0, |x-x_0| < a(t_0-t)\}. Теорема: Пусть u \in C^2(\mathbb{R}_+ \times \mathbb{R}^n) — решение однородной задачи Коши. Тогда для любого (t_0, x_0) и любого \tau \in (0, t_0) справедливо энергетическое неравенство: E(\tau) \le E(0) где энергия E(t) определяется как интеграл по сечению конуса: E(t) = \frac{1}{2} \int_{|x-x_0|<a(t_0-t)} (u_t^2(t,x) + a^2 |\nabla u(t,x)|^2) dx Идея доказательства: Аналогично доказательству энергетического равенства, интегрируется тождество (u_{tt} - a^2\Delta u)u_t = 0 по усеченному конусу K_\tau. После применения формулы Грина получается интеграл по границе конуса. Показывается, что интеграл по боковой поверхности конуса неотрицателен. Это приводит к неравенству E(\tau) - E(0) \le 0.

Теорема единственности: Решение задачи Коши для волнового уравнения в классе C^2 единственно. Доказательство: Пусть u_1, u_2 — два решения. Их разность w = u_1 - u_2 решает задачу Коши с нулевыми начальными данными. Тогда энергия E_w(0) = 0. Из энергетического неравенства следует, что E_w(t) \le E_w(0) = 0 для любого t в любой области зависимости. Так как E_w(t) \ge 0, то E_w(t) \equiv 0. Отсюда, как и в п. 5, следует, что w(t,x) \equiv 0. Следовательно, u_1 \equiv u_2.

7. Метод Фурье для волнового уравнения в пространстве размерности 2 (прямоугольник).

(Этот вопрос не разобран в лекциях. Приводится стандартное решение.)

Рассмотрим задачу для прямоугольной мембраны \Omega = [0, L_x] \times [0, L_y] с закрепленными краями. \begin{cases} u_{tt} = a^2(u_{xx} + u_{yy}) \\ u(0, x, y) = \varphi(x, y), \quad u_t(0, x, y) = \psi(x, y) \\ u|_{\partial\Omega} = 0 \end{cases} Ищем решение в виде u(t,x,y) = T(t)X(x)Y(y). Подстановка в уравнение и разделение переменных дают: \frac{T''(t)}{a^2T(t)} = \frac{X''(x)}{X(x)} + \frac{Y''(y)}{Y(y)} = -\lambda Это приводит к трем обыкновенным дифференциальным уравнениям: 1. X''(x) + \lambda_x X(x) = 0, \quad X(0)=0, X(L_x)=0 \implies \lambda_{x,n} = (\frac{n\pi}{L_x})^2, X_n(x) = \sin(\frac{n\pi x}{L_x}) 2. Y''(y) + \lambda_y Y(y) = 0, \quad Y(0)=0, Y(L_y)=0 \implies \lambda_{y,m} = (\frac{m\pi}{L_y})^2, Y_m(y) = \sin(\frac{m\pi y}{L_y}) 3. T_{nm}''(t) + a^2(\lambda_{x,n} + \lambda_{y,m})T_{nm}(t) = 0

Обозначим \omega_{nm} = a\sqrt{(\frac{n\pi}{L_x})^2 + (\frac{m\pi}{L_y})^2}. Решение для T(t): T_{nm}(t) = A_{nm} \cos(\omega_{nm}t) + B_{nm} \sin(\omega_{nm}t) Общее решение — это двойной ряд Фурье: u(t,x,y) = \sum_{n=1}^\infty \sum_{m=1}^\infty (A_{nm}\cos(\omega_{nm}t) + B_{nm}\sin(\omega_{nm}t)) \sin(\frac{n\pi x}{L_x})\sin(\frac{m\pi y}{L_y}) Коэффициенты A_{nm} и B_{nm} находятся из начальных условий с помощью двойных интегралов (коэффициенты двойного ряда Фурье).

8. Метод Фурье для волнового уравнения в пространстве размерности 2 (круг).

(Этот вопрос не разобран в лекциях. Приводится стандартное решение.)

Рассмотрим колебания круглой мембраны радиуса R с закрепленным краем. В полярных координатах (r, \theta) уравнение имеет вид: u_{tt} = a^2\left(u_{rr} + \frac{1}{r}u_r + \frac{1}{r^2}u_{\theta\theta}\right) Ищем решение в виде u(t, r, \theta) = T(t)V(r, \theta). Это приводит к уравнению Гельмгольца для V: \Delta V + k^2 V = 0, \quad k^2 = \lambda которое решается дальнейшим разделением переменных V(r, \theta) = R(r)\Theta(\theta).

  1. \Theta''(\theta) + m^2\Theta(\theta) = 0 \implies \Theta_m(\theta) = C_1\cos(m\theta) + C_2\sin(m\theta). Условие периодичности требует m \in \mathbb{Z}.
  2. r^2R''(r) + rR'(r) + (k^2r^2 - m^2)R(r) = 0. Это уравнение Бесселя. Граничное условие u(t, R, \theta)=0 дает R(R)=0. Решениями, которые не бесконечны в центре (r=0), являются функции Бесселя первого рода J_m(kr). Условие J_m(kR)=0 определяет дискретный набор допустимых значений k. Пусть \mu_{mn} — n-й положительный корень уравнения J_m(z)=0. Тогда k_{mn} = \mu_{mn}/R.
  3. Уравнение для T(t) имеет вид T'' + (ak_{mn})^2 T = 0.

Общее решение представляет собой двойной ряд по функциям Бесселя и тригонометрическим функциям.

9. Формула Кирхгофа для однородного волнового уравнения.

(Источник: Лекция 4)

Формула Кирхгофа дает решение задачи Коши для однородного волнового уравнения в трехмерном пространстве (n=3): \begin{cases} u_{tt} = a^2 \Delta u \\ u(0, x) = \varphi(x), \quad u_t(0, x) = \psi(x) \end{cases} Теорема (Формула Кирхгофа): Пусть \varphi \in C^3(\mathbb{R}^3), \psi \in C^2(\mathbb{R}^3). Тогда решение задачи Коши дается формулой: u(t, x) = \frac{1}{4\pi a^2 t} \int_{|\xi-x|=at} \psi(\xi) dS_\xi + \frac{\partial}{\partial t}\left(\frac{1}{4\pi a^2 t} \int_{|\xi-x|=at} \varphi(\xi) dS_\xi \right) Интегрирование ведется по сфере S_{at}(x) радиуса at с центром в точке x. Формулу можно переписать в виде: u(t,x) = t \cdot M_t[\psi] + \frac{\partial}{\partial t}(t \cdot M_t[\varphi]) где M_t[g] = \frac{1}{4\pi a^2 t^2}\int_{|\xi-x|=at} g(\xi) dS_\xi — среднее значение функции g по сфере.

Лемма (основа вывода): Для функции M_g(t,x) = \frac{1}{4\pi a^2 t} \int_{|\xi-x|=at} g(\xi) dS_\xi (с t в первой степени), где g \in C^2(\mathbb{R}^3), выполняются тождества: 1. \frac{\partial^2}{\partial t^2}M_g = a^2 \Delta M_g 2. M_g|_{t=0} = 0 3. \frac{\partial M_g}{\partial t}|_{t=0} = g(x) Из этой леммы следует, что u_1(t,x) = M_\psi(t,x) решает задачу с начальными условиями (0, \psi), а u_2(t,x) = \frac{\partial}{\partial t} M_\varphi(t,x) решает задачу с начальными условиями (\varphi, 0). По принципу суперпозиции их сумма дает искомое решение.

10. Метод спуска. Формула Пуассона для однородного волнового уравнения.

(Источник: Лекция 4, раздел 8)

Метод спуска — это способ получения решения задачи Коши в пространстве меньшей размерности из решения в пространстве большей размерности.

Рассмотрим задачу Коши в \mathbb{R}^2. Мы можем “поднять” ее в \mathbb{R}^3, считая, что начальные функции \varphi(x_1, x_2) и \psi(x_1, x_2) не зависят от третьей координаты x_3. Тогда и решение u(t, x_1, x_2, x_3) не будет зависеть от x_3. Для этого трехмерного решения мы можем применить формулу Кирхгофа.

u(t, x_1, x_2) = \frac{1}{4\pi a^2 t} \int_{S_{at}(x)} \psi(\xi_1, \xi_2) dS_\xi + \frac{\partial}{\partial t}\left(\frac{1}{4\pi a^2 t} \int_{S_{at}(x)} \varphi(\xi_1, \xi_2) dS_\xi \right) где x=(x_1,x_2,0), \xi=(\xi_1, \xi_2, \xi_3), а S_{at}(x) — сфера в \mathbb{R}^3. Интеграл по сфере можно “спустить” в интеграл по кругу D_{at}(x_1,x_2) в плоскости (\xi_1, \xi_2). Элемент площади сферы dS_\xi проецируется на плоскость с якобианом: dS_\xi = \frac{at}{\sqrt{a^2t^2 - r^2}}d\xi_1 d\xi_2, где r^2 = (\xi_1-x_1)^2 + (\xi_2-x_2)^2. Интеграл по сфере равен удвоенному интегралу по кругу (верхняя и нижняя полусферы дают одинаковый вклад).

После преобразований получается формула Пуассона для решения задачи Коши в \mathbb{R}^2: u(t, x, y) = \frac{1}{2\pi a} \int_{r \le at} \frac{\psi(\xi, \eta)}{\sqrt{a^2t^2 - r^2}} d\xi d\eta + \frac{\partial}{\partial t} \left( \frac{1}{2\pi a} \int_{r \le at} \frac{\varphi(\xi, \eta)}{\sqrt{a^2t^2 - r^2}} d\xi d\eta \right) где r^2 = (\xi-x)^2 + (\eta-y)^2.

11. Принцип Дюамеля. Формулы Кирхгофа и Пуассона для неоднородного волнового уравнения.

(Источник: Лекция 4, раздел 9; Лекция 5, раздел 10)

Теорема (Принцип Дюамеля): Решение задачи Коши для неоднородного уравнения с нулевыми начальными условиями: \begin{cases} u_{tt} = a^2 \Delta u + f(t,x) \\ u(0, x) = 0, \quad u_t(0, x) = 0 \end{cases} можно представить в виде: u(t, x) = \int_0^t U(t-\tau, x; \tau) d\tau где U(t, x; \tau) — решение задачи Коши для однородного уравнения с начальными данными, “заданными” в момент времени \tau: \begin{cases} U_{tt} = a^2 \Delta U \\ U(0, x; \tau) = 0, \quad U_t(0, x; \tau) = f(\tau, x) \end{cases} (Примечание: в лекциях приведена немного другая, но эквивалентная формулировка U(t-\tau, x; \tau)).

Используя этот принцип, можно дописать интегральные члены к формулам Кирхгофа и Пуассона, чтобы учесть неоднородность f(t,x).

Формула Кирхгофа для неоднородного уравнения (n=3): u(t,x) = u_{hom}(t,x) + \frac{1}{4\pi a^2} \int_0^t \frac{1}{t-\tau} \int_{|\xi-x|=a(t-\tau)} f(\tau, \xi) dS_\xi d\tau

Формула Пуассона для неоднородного уравнения (n=2): u(t,x) = u_{hom}(t,x) + \frac{1}{2\pi a} \int_0^t \int_{|\xi-x|\le a(t-\tau)} \frac{f(\tau, \xi)}{\sqrt{a^2(t-\tau)^2 - |\xi-x|^2}} d\xi d\tau Здесь u_{hom}(t,x) — соответствующее решение однородного уравнения (формулы из п. 9 и 10).

12. Распространение волн в пространствах различной размерности. Качественные различия.

(Источник: Лекция 5, раздел 11)

Рассмотрим решение задачи Коши с финитными начальными данными, т.е. \varphi(x) и \psi(x) равны нулю вне некоторого шара |x| \le R.

Случай n=3 (трехмерное пространство): Решение дается формулой Кирхгофа. Интегрирование идет по сфере |\xi-x|=at. - Если at < |x|-R (сфера интегрирования не достигла области возмущения), то u(t,x)=0. - Если |x|-R < at < |x|+R (сфера пересекает область возмущения), то u(t,x) может быть не равно нулю. - Если at > |x|+R (сфера прошла область возмущения), то u(t,x)=0 снова.

Это означает, что в точке x возмущение приходит в момент времени t_1 = (|x|-R)/a и полностью прекращается в момент t_2 = (|x|+R)/a. Волна имеет четкий передний и задний фронт. Это называется принципом Гюйгенса. Искажения сигнала не происходит.

Случай n=2 (двумерное пространство): Решение дается формулой Пуассона. Интегрирование идет по кругу |\xi-x| \le at. - Если at < |x|-R (круг интегрирования не достиг области возмущения), то u(t,x)=0. - Как только at > |x|-R, круг начинает пересекать область возмущения, и u(t,x) становится отличным от нуля.

Волна имеет четкий передний фронт, но задний фронт размыт. После прохождения переднего фронта возмущение в точке x не прекращается, а лишь затухает со временем, так как область возмущения навсегда остается внутри области интегрирования. Происходит диффузия волны. Принцип Гюйгенса не выполняется.

Случай n=1 (одномерное пространство): Решение дается формулой Даламбера. Оно состоит из двух частей: 1. Слагаемое с \varphi: \frac{1}{2}(\varphi(x+at) + \varphi(x-at)). Оно ведет себя как в 3D — имеет четкий передний и задний фронт. 2. Слагаемое с \psi: \frac{1}{2a}\int_{x-at}^{x+at}\psi(s)ds. Оно ведет себя как в 2D — имеет четкий передний фронт и размытый задний.

Таким образом, в 1D поведение зависит от типа начального возмущения. Если струну оттянуть и отпустить (\psi=0), возмущение будет иметь четкие фронты. Если по струне ударить (\varphi=0), звучание будет затухающим, с размытым задним фронтом.

13. Пространства основных функций \mathcal{D}(\Omega), примеры основных функций. Действия над основными функциями. Сходимость основных функций, примеры сходящихся и расходящихся последовательностей.

(Источник: Лекция 6, раздел 13)

Пространство основных функций \mathcal{D}(\Omega) (или C_0^\infty(\Omega)) — это множество функций \varphi(x), определенных в области \Omega \subset \mathbb{R}^n, которые удовлетворяют двум условиям: 1. Бесконечная гладкость: \varphi \in C^\infty(\Omega). 2. Финитность: носитель функции \text{supp}(\varphi) = \overline{\{x \in \Omega : \varphi(x) \neq 0\}} является компактом, лежащим в \Omega.

Пример основной функции (“шапочка”): \varphi(x) = \begin{cases} e^{-\frac{1}{1-x^2}}, & |x| < 1 \\ 0, & |x| \ge 1 \end{cases} Эта функция бесконечно дифференцируема на всей оси \mathbb{R}, а её носитель — отрезок [-1, 1].

Действия над основными функциями: - Линейные операции: Если \varphi, \psi \in \mathcal{D}(\Omega), то \alpha\varphi + \beta\psi \in \mathcal{D}(\Omega). - Умножение на C^\infty-функцию: Если \varphi \in \mathcal{D}(\Omega) и f \in C^\infty(\Omega), то f\varphi \in \mathcal{D}(\Omega). - Дифференцирование: Если \varphi \in \mathcal{D}(\Omega), то любая её производная D^\alpha \varphi также принадлежит \mathcal{D}(\Omega).

Сходимость в \mathcal{D}(\Omega): Последовательность \varphi_n \in \mathcal{D}(\Omega) сходится к \varphi \in \mathcal{D}(\Omega) при n \to \infty (пишется \varphi_n \to \varphi в \mathcal{D}(\Omega)), если: 1. Существует компакт K \subset \Omega такой, что \text{supp}(\varphi_n) \subset K для всех n. (Условие равностепенной финитности). 2. Для любого мультииндекса \alpha, последовательность производных D^\alpha \varphi_n равномерно сходится к D^\alpha \varphi на K (а значит и на \Omega).

Примеры последовательностей: - Сходящаяся: \varphi_n(x) = \frac{1}{n}\varphi(x), где \varphi(x) — “шапочка”. Здесь \varphi_n \to 0 в \mathcal{D}(\mathbb{R}). Носитель у всех функций один и тот же, и все производные равномерно сходятся к нулю. - Расходящаяся (нарушение условия 1): \varphi_n(x) = \varphi(x-n). Это последовательность “убегающих шапочек”. Нет общего компакта, содержащего все носители. - Расходящаяся (нарушение условия 2): \varphi_n(x) = \sin(nx)\varphi(x). Хотя носитель общий, производные D\varphi_n = n\cos(nx)\varphi(x) + \sin(nx)\varphi'(x) не сходятся равномерно (из-за множителя n).

14. Пространства обобщенных функций \mathcal{D}'(\Omega). Примеры: регулярные обобщенные функции, \delta-функция. Действия над обобщенными функциями. Сходимость обобщенных функций.

(Источник: Лекция 6, разделы 13, 14)

Пространство обобщенных функций (распределений) \mathcal{D}'(\Omega) — это пространство всех непрерывных линейных функционалов на пространстве основных функций \mathcal{D}(\Omega). Элемент f \in \mathcal{D}'(\Omega) — это отображение f: \mathcal{D}(\Omega) \to \mathbb{R}, такое что: 1. Линейность: \langle f, \alpha\varphi + \beta\psi \rangle = \alpha\langle f, \varphi \rangle + \beta\langle f, \psi \rangle. 2. Непрерывность: Если \varphi_n \to \varphi в \mathcal{D}(\Omega), то \langle f, \varphi_n \rangle \to \langle f, \varphi \rangle в \mathbb{R}. Здесь \langle f, \varphi \rangle обозначает действие функционала f на функцию \varphi.

Примеры: 1. Регулярные обобщенные функции: Любая локально интегрируемая функция f(x) \in L^1_{loc}(\Omega) порождает обобщенную функцию (которую обозначают той же буквой f) по правилу: \langle f, \varphi \rangle = \int_\Omega f(x)\varphi(x) dx Линейность очевидна, непрерывность доказывается с использованием свойств сходимости в \mathcal{D}(\Omega). 2. \delta-функция Дирака: Это сингулярная (нерегулярная) обобщенная функция, определяемая как: \langle \delta_{x_0}, \varphi \rangle = \varphi(x_0) (Обычно пишут \delta(x-x_0), а при x_0=0 просто \delta(x)). Она “измеряет” значение основной функции в точке x_0.

Действия над обобщенными функциями: Действия определяются по принципу дуальности, перенося операцию с обобщенной функции на основную. - Сложение и умножение на число: \langle \alpha f + \beta g, \varphi \rangle = \alpha \langle f, \varphi \rangle + \beta \langle g, \varphi \rangle - Умножение на C^\infty-функцию: Пусть a(x) \in C^\infty(\Omega). \langle af, \varphi \rangle = \langle f, a\varphi \rangle (Это корректно, так как a\varphi \in \mathcal{D}(\Omega)). - Дифференцирование: (см. следующий вопрос).

Сходимость в \mathcal{D}'(\Omega): Последовательность f_n \in \mathcal{D}'(\Omega) сходится к f \in \mathcal{D}'(\Omega) (слабая сходимость), если для любой основной функции \varphi \in \mathcal{D}(\Omega) числовая последовательность \langle f_n, \varphi \rangle сходится к \langle f, \varphi \rangle: f_n \to f \text{ в } \mathcal{D}'(\Omega) \iff \forall \varphi \in \mathcal{D}(\Omega), \quad \lim_{n\to\infty} \langle f_n, \varphi \rangle = \langle f, \varphi \rangle

15. Дифференцирование обобщенных функций. \delta-функция как производная функции Хевисайда. Производные \delta-функции.

(Источник: Лекция 6, раздел 14)

Определение производной: Производная D^\alpha f обобщенной функции f \in \mathcal{D}'(\Omega) определяется по формуле: \langle D^\alpha f, \varphi \rangle = (-1)^{|\alpha|} \langle f, D^\alpha \varphi \rangle, \quad \forall \varphi \in \mathcal{D}(\Omega) Это определение мотивировано формулой интегрирования по частям для регулярных функций. Любая обобщенная функция бесконечно дифференцируема в смысле этого определения.

\delta-функция как производная функции Хевисайда: Функция Хевисайда: \Theta(x) = \begin{cases} 1, & x > 0 \\ 0, & x \le 0 \end{cases} Она является регулярной обобщенной функцией. Найдем ее производную \Theta': \langle \Theta', \varphi \rangle = (-1)^1 \langle \Theta, \varphi' \rangle = -\int_{-\infty}^\infty \Theta(x)\varphi'(x)dx = -\int_0^\infty \varphi'(x)dx Так как \varphi(x) финитна, она равна нулю при больших x. -\int_0^\infty \varphi'(x)dx = -[\varphi(x)]_0^\infty = -(\lim_{x\to\infty}\varphi(x) - \varphi(0)) = -(0 - \varphi(0)) = \varphi(0) Мы получили, что \langle \Theta', \varphi \rangle = \varphi(0) = \langle \delta, \varphi \rangle. Следовательно, \Theta' = \delta.

Производные \delta-функции: Используя общее определение, производная \delta-функции k-го порядка определяется как: \langle \delta^{(k)}, \varphi \rangle = (-1)^k \langle \delta, \varphi^{(k)} \rangle = (-1)^k \varphi^{(k)}(0) Например: - \langle \delta', \varphi \rangle = -\varphi'(0). \delta' описывает диполь. - \langle \delta'', \varphi \rangle = \varphi''(0). \delta'' описывает квадруполь.

16. Определение фундаментального решения дифференциального уравнения. Фундаментальное решения уравнения Лапласа.

(Источник: Лекция 7, раздел 15)

Определение: Пусть L — линейный дифференциальный оператор. Фундаментальным решением уравнения Lu=0 называется обобщенная функция E(x, x_0) (или просто E(x) для операторов с постоянными коэффициентами, с x_0=0), которая удовлетворяет уравнению: L E(x, x_0) = \delta(x-x_0) Здесь L действует на E в смысле теории обобщенных функций. Фундаментальное решение — это, по сути, “отклик” системы на точечное возмущение, описываемое \delta-функцией.

Фундаментальное решение уравнения Лапласа: Уравнение Лапласа: \Delta u = 0. Мы ищем решение E уравнения Пуассона \Delta E = \delta(x). Такое решение (с точностью до выбора констант) имеет вид: - Для n=2 (в плоскости): E(x) = \frac{1}{2\pi} \ln |x| - Для n > 2 (в пространстве): E(x) = \frac{1}{(2-n)\omega_n |x|^{n-2}} = -\frac{\Gamma(n/2)}{2(n-2)\pi^{n/2}|x|^{n-2}} где \omega_n — площадь поверхности единичной сферы в \mathbb{R}^n. В частности, для n=3: E(x) = -\frac{1}{4\pi |x|} Эти функции являются локально интегрируемыми и, следовательно, регулярными обобщенными функциями. В лекции доказывается, что \Delta E = \delta(x), применяя теорему о представлении функции через потенциалы (формулу Грина).

17. Построение фундаментального решения обыкновенного линейного дифференциального уравнения.

(Источник: Лекция 7, раздел 16)

Рассмотрим обыкновенное линейное дифференциальное уравнение с постоянными коэффициентами: L(u) = u^{(m)} + a_{m-1}u^{(m-1)} + \dots + a_0 u = 0 Мы ищем фундаментальное решение E(x), удовлетворяющее L(E) = \delta(x).

Теорема: Фундаментальным решением является функция E(x) = \Theta(x)Z(x), где: - \Theta(x) — функция Хевисайда. - Z(x) — решение однородного уравнения L(Z)=0, удовлетворяющее специальным начальным условиям в точке x=0: Z(0) = Z'(0) = \dots = Z^{(m-2)}(0) = 0, \quad Z^{(m-1)}(0) = 1

Идея доказательства: Используется формула для производной произведения и формула для производной разрывной функции (следствие из леммы в лекциях): f^{(k)} = \{f^{(k)}\} + [f]_0 \delta^{(k-1)} + [f']_0 \delta^{(k-2)} + \dots + [f^{(k-1)}]_0 \delta где \{f^{(k)}\} — обычная производная (где существует), а [f^{(j)}]_0 — скачок j-й производной в нуле.

Для E(x) = \Theta(x)Z(x), все скачки до (m-2)-й производной включительно равны нулю из-за начальных условий на Z. Скачок (m-1)-й производной равен Z^{(m-1)}(0) = 1. Применяя оператор L к E(x), все члены, содержащие производные от \delta, сокращаются, и остается только один член с \delta(x), который появляется от E^{(m)}(x). Этот член равен 1 \cdot \delta(x). Члены без \delta-функций дают L(\Theta(x)Z(x)) в обычном смысле, что равно \Theta(x)L(Z(x))=0 для x>0. Таким образом, L(E) = \delta(x).

18. Понятие обобщенной производной. Связь обобщенной производной и производной в смысле теории обобщенных функций. Пространства Соболева H^1(\Omega) и \mathring{H}^1(\Omega), их полнота.

(Источник: Лекция 8, раздел 17)

Обобщенная (слабая) производная: Пусть u \in L^1_{loc}(\Omega). Функция v \in L^1_{loc}(\Omega) называется обобщенной производной D^\alpha u функции u, если для любой основной функции \varphi \in \mathcal{D}(\Omega) выполняется равенство: \int_\Omega u(x) D^\alpha\varphi(x) dx = (-1)^{|\alpha|} \int_\Omega v(x)\varphi(x) dx Если такая функция v существует, она единственна (в смысле L^1_{loc}).

Связь с производной в \mathcal{D}': Обобщенная производная — это частный случай производной в смысле теории обобщенных функций, когда и сама функция, и ее производная являются регулярными обобщенными функциями. Если v = D^\alpha u в смысле обобщенных производных, то \langle D^\alpha u, \varphi \rangle = \langle v, \varphi \rangle.

Пространства Соболева: - H^1(\Omega) — это пространство функций u \in L^2(\Omega), все обобщенные производные первого порядка которых также принадлежат L^2(\Omega). H^1(\Omega) = \{u \in L^2(\Omega) : \frac{\partial u}{\partial x_i} \in L^2(\Omega), i=1,\dots,n\}. Это гильбертово пространство со скалярным произведением и нормой: (u,v)_1 = \int_\Omega (uv + \nabla u \cdot \nabla v)dx, \quad \|u\|_1 = \sqrt{(u,u)_1} - \mathring{H}^1(\Omega) — это замыкание пространства основных функций \mathcal{D}(\Omega) (или C_0^\infty(\Omega)) в норме пространства H^1(\Omega). Интуитивно, это пространство функций из H^1(\Omega), которые “равны нулю на границе”.

Полнота: Оба пространства, H^1(\Omega) и \mathring{H}^1(\Omega) (как замкнутое подпространство гильбертова пространства), являются полными, то есть гильбертовыми.

19. Неравенство Фридрихса. Эквивалентные нормы в пространстве \mathring{H}^1(\Omega).

(Источник: Лекция 8, раздел 19)

Лемма (Неравенство Фридрихса): Пусть \Omega \subset \mathbb{R}^n — ограниченная область. Тогда существует константа C = C(\Omega) > 0 такая, что для любой функции u \in \mathring{H}^1(\Omega) выполняется неравенство: \int_\Omega u^2(x) dx \le C(\Omega) \int_\Omega |\nabla u(x)|^2 dx Или в нормах: \|u\|_{L^2}^2 \le C \|\nabla u\|_{L^2}^2.

Идея доказательства (для u \in C_0^\infty(\Omega)): Помещаем \Omega в достаточно большой куб K_R = [-R, R]^n. Для любой точки x=(x_1, x') можно написать: u(x_1, x') = \int_{-R}^{x_1} \frac{\partial u}{\partial t}(t, x') dt. Применяя неравенство Коши-Шварца: u^2(x) \le \left(\int_{-R}^{x_1} 1^2 dt\right) \left(\int_{-R}^{x_1} (\frac{\partial u}{\partial t})^2 dt\right) \le 2R \int_{-R}^R (\frac{\partial u}{\partial x_1})^2 dx_1. Интегрируя это неравенство по всему кубу K_R, получаем требуемый результат. Для произвольной функции из \mathring{H}^1(\Omega) результат следует из плотности C_0^\infty(\Omega).

Эквивалентные нормы в \mathring{H}^1(\Omega): В пространстве \mathring{H}^1(\Omega) норма \|u\|_2 = \left(\int_\Omega |\nabla u|^2 dx\right)^{1/2} эквивалентна стандартной норме \|u\|_1 = \left(\int_\Omega (u^2 + |\nabla u|^2) dx\right)^{1/2}. Доказательство: 1. Очевидно, \|u\|_2 \le \|u\|_1. 2. Из неравенства Фридрихса: \|u\|_1^2 = \|u\|_{L^2}^2 + \|u\|_2^2 \le C\|u\|_2^2 + \|u\|_2^2 = (C+1)\|u\|_2^2. Следовательно, \|u\|_1 \le \sqrt{C+1} \|u\|_2.

Таким образом, \|u\|_2 является полноценной нормой в \mathring{H}^1(\Omega), и она порождает скалярное произведение (u,v)_2 = \int_\Omega \nabla u \cdot \nabla v \, dx.

20. Определение обобщенного решения первой краевой задачи для уравнения Пуассона. Связь классического и обобщенного решений.

(Источник: Лекция 8, раздел 18, 20)

Рассмотрим первую краевую задачу для уравнения Пуассона: \begin{cases} -\Delta u = f(x), & x \in \Omega \\ u|_{\partial\Omega} = 0 \end{cases} (Знак минус удобен для дальнейших формулировок).

Классическое решение: Функция u \in C^2(\Omega) \cap C(\bar{\Omega}), удовлетворяющая уравнению и граничным условиям в обычном смысле.

Обобщенное (слабое) решение: Функция u \in \mathring{H}^1(\Omega) называется обобщенным решением, если для любой “пробной” функции v \in \mathring{H}^1(\Omega) (достаточно проверять для v \in C_0^\infty(\Omega)) выполняется интегральное тождество: \int_\Omega \nabla u \cdot \nabla v \, dx = \int_\Omega f v \, dx Или в терминах скалярного произведения в \mathring{H}^1(\Omega): (u, v)_2 = (f, v)_{L^2}.

Связь решений: 1. Если u — классическое решение (достаточно гладкое), то оно является и обобщенным решением. Доказательство: Умножим уравнение -\Delta u = f на пробную функцию v \in C_0^\infty(\Omega) и проинтегрируем по \Omega. -\int_\Omega (\Delta u) v \, dx = \int_\Omega fv \, dx. Применяя формулу Грина к левой части и учитывая, что v=0 на \partial\Omega: \int_\Omega \nabla u \cdot \nabla v \, dx - \int_{\partial\Omega} \frac{\partial u}{\partial \nu} v \, dS = \int_\Omega fv \, dx \implies \int_\Omega \nabla u \cdot \nabla v \, dx = \int_\Omega fv \, dx. Тождество выполнено.

  1. Если u — обобщенное решение, и при этом оно достаточно гладкое (u \in C^2(\Omega)), то оно является и классическим решением. Доказательство: Проводим рассуждения в обратную сторону. Из \int_\Omega \nabla u \cdot \nabla v \, dx = \int_\Omega fv \, dx, применяя формулу Грина, получаем \int_\Omega (-\Delta u - f)v \, dx = 0 для любой v \in C_0^\infty(\Omega). В силу основной леммы вариационного исчисления, отсюда следует, что -\Delta u - f = 0 поточечно.

21. Теорема существования обобщенного решения уравнения Пуассона.

(Источник: Лекция 8, раздел 21)

Теорема: Для любой функции f \in L^2(\Omega) первая краевая задача для уравнения Пуассона \begin{cases} -\Delta u = f(x), & x \in \Omega \\ u|_{\partial\Omega} = 0 \end{cases} имеет единственное обобщенное решение u \in \mathring{H}^1(\Omega).

Доказательство: Доказательство элегантно и опирается на фундаментальную теорему функционального анализа. 1. Рассмотрим линейный функционал на гильбертовом пространстве \mathring{H}^1(\Omega) (со скалярным произведением (u,v)_2 = \int_\Omega \nabla u \cdot \nabla v \, dx): L(v) = \int_\Omega f v \, dx = (f, v)_{L^2} 2. Покажем, что этот функционал непрерывен (ограничен). По неравенству Коши-Шварца в L^2 и неравенству Фридрихса: |L(v)| = |(f, v)_{L^2}| \le \|f\|_{L^2} \|v\|_{L^2} \le \|f\|_{L^2} \sqrt{C} \|\nabla v\|_{L^2} = (\sqrt{C}\|f\|_{L^2}) \|v\|_2 Так как |L(v)| \le K \|v\|_2, функционал L является ограниченным (непрерывным) на \mathring{H}^1(\Omega). 3. Применяем теорему Рисса о представлении линейных функционалов. Согласно этой теореме, для любого непрерывного линейного функционала L на гильбертовом пространстве H существует единственный элемент u \in H такой, что L(v) = (u, v)_H для всех v \in H. 4. В нашем случае H=\mathring{H}^1(\Omega) со скалярным произведением (\cdot, \cdot)_2. Следовательно, существует единственный элемент u \in \mathring{H}^1(\Omega) такой, что: L(v) = (u, v)_2 для всех v \in \mathring{H}^1(\Omega). 5. Расписывая, получаем: \int_\Omega fv \, dx = \int_\Omega \nabla u \cdot \nabla v \, dx Это в точности определение обобщенного решения. Существование и единственность доказаны.
